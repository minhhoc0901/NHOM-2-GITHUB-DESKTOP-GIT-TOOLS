BOOK API PROGRESS

Module: Book Management

Planned APIs:
1. List book
2. Delete book
List Book API
- Method: GET
- Endpoint: /api/books
- Description: return all books
- Response:
  + id
  + title
  + author
  + price
Delete Book API
- Method: DELETE
- Endpoint: /api/books/{id}
- Description: delete a book by id
